# Receipt Scanner App

Full-stack receipt scanning application that extracts receipt data using OCR and exports it to Google Sheets.

## Tech Stack

**Backend**

* Java Spring Boot
* Supabase (DB, Storage, Auth)
* Google Vision OCR
* Google Sheets API

**Frontend**

* (React / Next / Vue — TBD)

## Project Structure

/backend – Spring Boot API
/frontend – Client app
/docs – Architecture + API documentation

## Core Flow

Receipt Upload → OCR → Data Extraction → Database → Google Sheets
